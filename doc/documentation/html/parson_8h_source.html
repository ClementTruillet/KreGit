103 /* Validation
104  This is *NOT* JSON Schema. It validates json by checking if object have identically
105  named fields with matching types.
106  For example schema {"name":"", "age":0} will validate
107  {"name":"Joe", "age":25} and {"name":"Joe", "age":25, "gender":"m"},
108  but not {"name":"Joe"} or {"name":"Joe", "age":"Cucumber"}.
109  In case of arrays, only first value in schema is checked against all values in tested array.
110  Empty objects ({}) validate all objects, empty arrays ([]) validate all arrays,
111  null validates values of every type.
112  */
113 JSON_Status json_validate(const JSON_Value *schema, const JSON_Value *value);
